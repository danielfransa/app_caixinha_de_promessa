# Politica de Privacidade - Caixinha de Promessas

Ultima atualizacao: 2 de maio de 2026

## 1. Introducao

O aplicativo `Caixinha de Promessas` foi criado para oferecer ao usuario uma experiencia simples de leitura de promessas biblicas de forma offline.

Esta politica de privacidade explica como o aplicativo trata dados e informacoes.

## 2. Coleta de dados

O aplicativo `Caixinha de Promessas`, em sua versao atual:

- nao exige cadastro;
- nao exige login;
- nao coleta dados pessoais;
- nao coleta localizacao;
- nao acessa contatos;
- nao acessa fotos, camera ou microfone;
- nao utiliza anuncios;
- nao utiliza rastreamento de usuarios.

## 3. Funcionamento do aplicativo

O aplicativo funciona localmente no dispositivo do usuario e utiliza apenas um arquivo interno com promessas biblicas para realizar o sorteio das mensagens exibidas.

## 4. Compartilhamento de dados

O aplicativo nao compartilha dados pessoais com terceiros, porque nao realiza coleta de dados pessoais.

## 5. Armazenamento e seguranca

Como o aplicativo nao coleta nem transmite dados pessoais, nao ha armazenamento externo de dados do usuario em servidores proprios para a funcionalidade atual do app.

## 6. Publico

O aplicativo pode ser utilizado por publico em geral e nao foi desenvolvido para realizar tratamento de dados pessoais sensiveis.

## 7. Alteracoes nesta politica

Esta politica pode ser atualizada no futuro caso o aplicativo passe a incluir novas funcionalidades, como compartilhamento, favoritos em nuvem, login ou notificacoes com tratamento de dados adicionais.

Sempre que houver alteracoes relevantes, a politica de privacidade sera revisada.

## 8. Contato

Para duvidas, sugestoes ou solicitacoes relacionadas a esta politica, entre em contato pelo email do desenvolvedor informado na pagina do app na Google Play.
