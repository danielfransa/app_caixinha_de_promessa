# Política de Privacidade - Caixinha de Promessas

Última atualização: 2 de maio de 2026

## 1. Introdução

O aplicativo **Caixinha de Promessas** foi criado para oferecer ao usuário uma experiência simples de leitura de promessas bíblicas de forma offline.

Esta Política de Privacidade explica como o aplicativo trata dados e informações.

## 2. Coleta de dados

O aplicativo **Caixinha de Promessas**, em sua versão atual:

- não exige cadastro;
- não exige login;
- não coleta dados pessoais;
- não coleta localização;
- não acessa contatos;
- não acessa fotos, câmera ou microfone;
- não utiliza anúncios;
- não utiliza rastreamento de usuários.

## 3. Funcionamento do aplicativo

O aplicativo funciona localmente no dispositivo do usuário e utiliza apenas um arquivo interno com promessas bíblicas para realizar o sorteio das mensagens exibidas.

## 4. Compartilhamento de dados

O aplicativo não compartilha dados pessoais com terceiros, porque não realiza coleta de dados pessoais.

## 5. Armazenamento e segurança

Como o aplicativo não coleta nem transmite dados pessoais, não há armazenamento externo de dados do usuário em servidores próprios para a funcionalidade atual do app.

## 6. Público

O aplicativo pode ser utilizado pelo público em geral e não foi desenvolvido para realizar tratamento de dados pessoais sensíveis.

## 7. Alterações nesta política

Esta política pode ser atualizada no futuro caso o aplicativo passe a incluir novas funcionalidades, como compartilhamento, favoritos em nuvem, login ou notificações com tratamento de dados adicionais.

Sempre que houver alterações relevantes, a política de privacidade será revisada.

## 8. Contato

Para dúvidas, sugestões ou solicitações relacionadas a esta política, entre em contato pelo email do desenvolvedor informado na página do app na Google Play.
