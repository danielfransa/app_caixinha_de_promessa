# Política de Privacidade - Caixinha de Promessas

Última atualização: 4 de maio de 2026

## 1. Introdução

O aplicativo **Caixinha de Promessas** foi criado com o objetivo de oferecer uma experiência simples, leve e acolhedora de leitura de promessas bíblicas, funcionando principalmente de forma offline no dispositivo do usuário.

Esta Política de Privacidade explica, de forma clara, como o aplicativo trata informações, dados e recursos do dispositivo.

## 2. Coleta de dados

Na versão atual, o aplicativo **Caixinha de Promessas**:

- não exige cadastro;
- não exige login;
- não coleta dados pessoais;
- não coleta dados de localização;
- não acessa contatos;
- não acessa câmera nem microfone;
- não utiliza anúncios;
- não utiliza cookies;
- não realiza rastreamento de usuários;
- não envia dados pessoais para servidores externos para o funcionamento principal do app.

## 3. Funcionamento do aplicativo

O aplicativo funciona localmente no dispositivo do usuário e utiliza arquivos internos para carregar e sortear promessas bíblicas.

No estado atual do projeto, o app não depende de conta, sincronização em nuvem nem serviços externos para exibir o conteúdo principal.

## 4. Compartilhamento de conteúdo pelo usuário

O aplicativo oferece recursos de:

- copiar o texto da promessa sorteada;
- compartilhar o texto com outros aplicativos;
- compartilhar uma imagem do card da promessa;
- salvar a imagem do card na galeria do dispositivo.

Essas ações só acontecem quando iniciadas voluntariamente pelo próprio usuário.

O aplicativo não envia automaticamente esse conteúdo a terceiros. Quando o usuário escolhe compartilhar uma promessa, o envio passa a ocorrer de acordo com o aplicativo externo selecionado, sujeito às políticas desse serviço externo.

## 5. Acesso a fotos e arquivos do dispositivo

Na versão atual, o aplicativo não realiza leitura da galeria pessoal do usuário para navegação ou análise de imagens.

Em alguns dispositivos Android, o app poderá solicitar permissão apenas para salvar localmente, na galeria do aparelho, a imagem do card gerado para o usuário.

Esse acesso, quando solicitado, é usado exclusivamente para gravar a imagem gerada pelo próprio aplicativo, mediante ação do usuário.

## 6. Compartilhamento de dados

O aplicativo não compartilha dados pessoais com terceiros, porque não realiza coleta de dados pessoais em sua funcionalidade atual.

Dados somente poderão ser compartilhados se houver obrigação legal aplicável.

## 7. Armazenamento e segurança

Como o aplicativo não coleta nem transmite dados pessoais para servidores externos em sua versão atual, não há base de armazenamento remoto de dados pessoais do usuário relacionada ao funcionamento principal do app.

Em algumas funcionalidades, como compartilhamento de imagem, o aplicativo pode gerar arquivos temporários locais no dispositivo apenas para concluir a ação solicitada pelo usuário.

Ainda assim, buscamos manter o projeto com boas práticas de desenvolvimento e uso responsável de recursos locais.

## 8. Links externos

O aplicativo pode abrir links externos, como o repositório do projeto no GitHub e a página de issues, quando o usuário escolhe essa ação na tela "Sobre".

Além disso, páginas associadas ao projeto, como site, repositório ou política de privacidade, podem conter links para serviços externos.

Esses serviços possuem políticas próprias, e o usuário deve consultá-las quando necessário.

## 9. Público

O aplicativo pode ser utilizado pelo público em geral e não foi desenvolvido para tratamento de dados pessoais sensíveis.

## 10. Alterações nesta política

Esta política poderá ser atualizada futuramente caso o aplicativo passe a incluir novas funcionalidades, como:

- favoritos sincronizados;
- login;
- notificações com integração externa;
- analytics;
- monetização;
- qualquer outro recurso que envolva tratamento adicional de dados.

Sempre que houver alterações relevantes, esta política será revisada.

## 11. Consentimento e uso continuado

Ao utilizar o aplicativo, o usuário declara estar ciente desta Política de Privacidade.

Caso não concorde com algum ponto, recomenda-se não utilizar o aplicativo.

## 12. Contato

Para dúvidas, sugestões ou solicitações relacionadas a esta Política de Privacidade, utilize o canal de contato informado na página pública do projeto ou na página do aplicativo na Google Play.
